Inside vs Outside List Style

list-style-position: inside;

  1. Lorem, ipsum.
  2. Similique, culpa.
  3. Atque, vel?
  4. Repudiandae, expedita.
  5. Perferendis, modi.

list-style-position: outside;

  1. Lorem, ipsum.
  2. Similique, culpa.
  3. Atque, vel?
  4. Repudiandae, expedita.
  5. Perferendis, modi.

Code Syntax

<style >
.inside {
      list-style-position: inside;
}

.outside {
      list-style-position: outside;
}
</style >
