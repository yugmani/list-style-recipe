Custom Cycling List

Code Syntax

<style >
ol {
      list-style:symbols(cyclic "♪" "♫" "♩" "♬");
}

@counter-style thumbs {
      system:cyclic;
      symbols:"♪" "♫" "♩" "♬";
      /*
      prefix: "before";
      */
      suffix: " ";
}

ul {
      list-style:thumbs;
}
</style >

Note: This is only supported in Firefox at the time of writing.


Reference: List Style Recipes: Chris Coyier on May 5, 2020
